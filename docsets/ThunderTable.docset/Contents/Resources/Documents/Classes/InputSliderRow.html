InputSliderRow

open class InputSliderRow: InputTableRow

A row which displays a UISlider and label displaying it’s value

Important

If you use the valueChangeHandler on this row, and want to use the sender part of that closure, make sure to use correctedValue rather than value due to the custom implementation of UISlider allowing for non-integer steps. This means sender.value !== value in that closure!
