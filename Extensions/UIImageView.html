  • set(imageURL:withPlaceholder:imageSize:animated:completion:)

    Loads an image from the given URL and upon completion updates self.image to reflect the returned image

    Once the image has loaded from the URL it will be displayed at the same size as the placeholderImage (or the provided size) so make sure that placeholderImage is the same size as the returned image. Not doing so causes strange frame behaviour in UITableViewCell subclasses

    Declaration

    Swift

    public func set(imageURL: URL?, withPlaceholder: UIImage?, imageSize: CGSize = CGSize.zero, animated: Bool = false, completion: ImageViewSetImageURLCompletion?)

    Parameters

    imageURL

    The URL to load the image from
    withPlaceholder

    The image to use as a placeholder for the loading image (Optional)
    imageSize

    The final image size that will be returned from the URL if known
    animated

    Whether the loading image should animate in when retrieved
    completion

    A closure which will be called upon the completed load of the image.
