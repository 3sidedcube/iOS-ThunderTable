Whether a value for the row is required before continuing

@property (nonatomic, assign) BOOL required

Discussion

If this is set to true the row will be returned by the TSCTableViewController method -missingRequiredInputRows if value is nil

Similarily it will cause TSCTableViewController method -isMissingRequiredInputRows to return true if value is nil
