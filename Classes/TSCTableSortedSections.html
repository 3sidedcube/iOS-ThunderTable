- (id)initWithItems:(NSArray *)items target:(id)target selector:(SEL)selector

Parameters

items

The array of TSCTableRows or objects that conform to the TSCTableRowDataSource protocol to be sorted
target

The object to send all selection events for rows inside the section
selector

The selector to call on the target when a row is selected

Discussion

Initializes the array with an array of TSCTableRows and sorts them into alphabetic order

target and selector can be set to nil if you wish to add a target to each individual row or object. The row object will be passsed to the method given in the selector if selector and target are specified
